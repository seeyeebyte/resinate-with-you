# Brand Name And Slogans

## Current Working Brand

Brand name:

```text
Resinate With You
```

Main slogan:

```text
Discover resin artists. Create pieces together.
```

Chinese positioning:

```text
一个帮助用户发现独立滴胶作者，并与作者产生定制、共创和直接联系的平台。
```

English positioning:

```text
A creator discovery and co-creation platform for handmade resin pieces and independent resin artists.
```

## Why This Direction Works

`Resinate With You` gives the project a warmer and more active feeling than a plain directory name.

It keeps the resin connection clear, but the phrase `With You` adds:

- co-creation
- custom work
- artist-fan interaction
- community feeling
- emotional connection

This fits the MVP because the platform still sends users to each artist's own shop or contact link, while the brand can suggest a more personal creative relationship.

## Brand Voice

The brand should feel:

- warm
- clean
- creative
- collaborative
- lightly curated
- not too marketplace-like
- not too empty-gallery-like

Use language around:

```text
discover
create together
custom pieces
artist stories
direct links
personal resin work
```

Avoid making version 1 sound like a full custom-order marketplace. The first version is still discovery and referral only.

## Naming Risk Notes

Quick public search notes from 2026-05-27:

- No exact strong result was found for `Resinate With You` as a resin artist discovery platform.
- Similar resin-related names and phrases exist, including `Resinate with Love`, `Resinate Custom Flooring`, `Resinate Flooring`, `Resinate Wood Co`, and other resin businesses using `Resinate`.
- `Resinate Custom Flooring` also uses the phrase `Where Our Floors Resinate With You`, which is close in wording but in a different flooring category.

Decision:

```text
Use Resinate With You as the working brand for the MVP.
```

Before public launch or paid promotion:

```text
Check domain availability.
Check Instagram, TikTok, Pinterest, Etsy, and Shopify handles.
Search USPTO and relevant trademark databases.
Consider a formal trademark clearance search if the project becomes commercial.
```

This document is not legal advice.
