# Visual Design Direction

## 1. 整体风格

关键词：

```text
简洁
精致
轻盈
有细节
有一点动画
商品和作者是主角
不要太像普通电商
不要太像艺术馆
```

目标感觉：

```text
一个精致的手作导流平台
像可浏览的精选目录
带一点收集拼图碎片的隐喻
```

## 2. 拼图概念

核心隐喻：

```text
每个作者是一块独特的拼图碎片。
每件作品也是一块小碎片。
平台把分散在 Instagram、Shopify、Etsy、个人网站上的碎片收集起来。
用户在这里找到适合自己的那一块。
```

可以用在：

- Logo 图形
- 首页背景细节
- 商品卡片 hover
- 作者卡片边角
- loading animation
- empty state
- category icon

不要做成：

- 儿童拼图风
- 过度卡通
- 大面积彩色拼图
- 太复杂的背景

## 3. 视觉语言

推荐：

```text
白底或柔和浅色底
大商品图
干净字体
清晰卡片
细线边框
少量柔和色块
轻微圆角
微妙阴影
```

拼图元素建议：

```text
不规则小切角
卡片边缘像拼图缺口一样轻微变化
背景里有很淡的 puzzle outline
作者头像周围用 puzzle piece frame
分类标签可以像小拼图片
```

## 4. 动画方向

轻动画即可：

```text
商品卡片 hover 微微上浮
图片 hover 稍微放大
筛选结果淡入
目录卡片进入视口时 stagger fade in
拼图碎片轻轻拼合成 logo
按钮 hover 有细微光泽
```

避免：

```text
大幅度飞行动画
过多滚动特效
影响加载速度的动画
让用户看不清商品的动画
```

## 5. 首页结构建议

第一屏：

```text
Logo + 导航
一句清楚 headline
一个大搜索框
下方露出横向滑动的商品/作者预览
```

第二屏：

```text
Favorite Finds This Week
3-6 个精选商品大卡片
```

第三屏：

```text
The Directory
搜索 + 分类 + 地区 + 是否接定制
作者卡片
```

第四屏：

```text
How it works
Discover -> Visit Artist Shop -> Buy Directly
```

第五屏：

```text
Apply to Join
作者入驻说明
```

## 6. 可以保留的参考模块

### Favorite Finds 模块

特点：

```text
大标题
3 列商品卡片
大图
商品名
作者名 + 地点
```

适合首页推荐商品。

### Directory 模块

特点：

```text
深色背景
大标题
搜索框
分类筛选
地点筛选
卡片列表
```

适合作者目录页或首页下半部分。

### 横向滑动预览模块

特点：

```text
大标题
中间搜索框
下方横向商品/品牌图片滑动
有动感但不复杂
```

适合首页第一屏底部露出一部分，让用户感觉“可以探索很多碎片”。

## 7. 推荐英文文案方向

Hero headline:

```text
Find the resin piece that fits.
```

Hero subtitle:

```text
Discover resin artists. Create pieces together.
```

Puzzle concept line:

```text
Every artist adds a different piece to the resin world.
```

Directory line:

```text
Search by style, location, product type, and custom availability.
```

Artist apply line:

```text
Add your piece to the directory.
```
